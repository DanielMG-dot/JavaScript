O que é DOM?

DOM significa Document Object Model

É o conjunto de de objetos dentro do navegador que dá acesso aos componentes internos do Website

Query Selector

Um jeito mais moderno de trabalhar com o DOM é com o comando "document.querySelector()" ou "document.querySelectorAll()"

Para selecionar os elementos pelo id: document.querySelector(div#prim)

Para selecionar os elementos pela classe: document.querySelector(p.teste)
